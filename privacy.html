개인정보처리방침

시행일: 2026년 4월 13일 | 최종 수정: 2026년 4월 13일

알바트래커(이하 "서비스")는 「개인정보 보호법」에 따라 이용자의 개인정보를 보호하고 이와 관련한 고충을 원활하게 처리할 수 있도록 다음과 같이 개인정보처리방침을 수립하여 공개합니다.

1. 수집하는 개인정보 항목 및 수집 방법

구분수집 항목수집 방법
Google 로그인이메일 주소, 이름, 프로필 사진Google 계정 연동 시 자동 수집
관리자 입력업체명, 사업주 정보(이름, 전화번호, 주소 — 근로계약서 작성 시), 직원 이름, 시급, 근무 시간서비스 이용 중 직접 입력
직원 입력이름, 근로계약서 응답 및 서명서비스 이용 중 직접 입력

서비스는 주민등록번호, 계좌번호 등 민감한 개인정보를 수집하지 않습니다.

2. 개인정보의 수집 및 이용 목적

  • 근무 시간 관리 및 급여 계산
  • 관리자-직원 간 업무 연동 (근로계약서, 급여명세서, 근무시간 알림)
  • 서비스 이용자 식별 및 인증

3. 개인정보의 보유 및 이용 기간

이용자의 개인정보는 서비스 이용 기간 동안 보유하며, 다음의 경우 지체 없이 파기합니다:

  • 이용자가 데이터 삭제를 요청한 경우
  • 관리자가 업체를 삭제한 경우 (삭제 예약 후 24시간 경과 시 자동 파기)
  • 직원이 업체에서 퇴사 처리한 경우

4. 개인정보의 제3자 제공

서비스는 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 서비스 운영을 위해 다음의 클라우드 인프라를 이용합니다:

서비스목적보관 위치
Google Firebase (Firestore)데이터 저장 및 실시간 동기화Google Cloud 서버
Google Firebase (Authentication)사용자 인증Google Cloud 서버
GitHub Pages / Firebase Hosting웹 애플리케이션 호스팅미국

위 서비스 제공업체는 데이터 전송 및 저장 시 암호화를 적용하며, 각 업체의 개인정보처리방침에 따라 관리됩니다.

5. 개인정보의 파기 절차 및 방법

  • 직원: 앱 내 "내 정보" 탭에서 퇴사 처리 또는 데이터 삭제 요청 가능
  • 관리자: 설정 탭에서 업체 삭제 가능 (24시간 유예 후 영구 삭제)
  • 삭제된 데이터는 Firebase Firestore에서 즉시 제거되며, 복구할 수 없습니다.

6. 이용자의 권리와 행사 방법

이용자는 언제든지 다음의 권리를 행사할 수 있습니다:

  • 개인정보 열람 요청
  • 개인정보 수정 요청
  • 개인정보 삭제 요청
  • 처리 정지 요청

위 권리는 앱 내 기능을 통해 직접 행사하거나, 아래 연락처로 요청하실 수 있습니다.

7. 개인정보의 안전성 확보 조치

  • 데이터 전송 시 SSL/TLS 암호화 적용
  • Firebase Firestore 저장 시 자동 암호화
  • Google OAuth 2.0 기반 인증으로 비밀번호 미저장
  • Firebase Security Rules를 통한 데이터 접근 제어
